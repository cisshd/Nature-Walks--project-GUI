Nature Walks is a multi-page GUI web project designed to help users discover peaceful walking trails and suggest new routes for others to explore.

##  Project Overview
The website provides information about nature walking trails, mainly focusing on locations in Najran and other regions in Saudi Arabia.  
Users can browse trails, compare them, and submit suggestions for new walking paths.

##  Features
- Multi-page website with clear navigation
- Trail listings with images and descriptions
- Trail comparison table (length & difficulty)
- Suggest a Trail form
- Responsive and user-friendly design
- Nature-inspired color theme

##  Technologies Used
- HTML5
- CSS3

##  Pages Included
- Home
- Trails
- Suggest a Trail
- About
- Enhancements

##  Purpose
This project was developed as part of a GUI course assignment to practice:
- Web page layout and structure
- Styling with CSS
- Forms, tables, and lists
- Creating a consistent and calm user interface

##  Future Improvements
- Add JavaScript for form validation
- Interactive filters for trail difficulty
- Map integration for trail locations
