Reflection

Working on this project gave me a practical opportunity to apply what I learned about HTML and CSS throughout the course. I enjoyed designing each page and seeing how small styling changes could significantly improve the overall user experience.

One of the biggest challenges I faced was aligning and organizing content consistently across multiple pages. Getting the layout to look clean, especially when using elements like tables and forms, required a lot of testing and adjusting margins, padding, and borders. I also spent extra time choosing suitable color combinations and ensuring that the design felt calm and nature-inspired.

If I were to improve the project further, I would like to explore using JavaScript to make the form more dynamic and add interactive elements like filters for trail types or difficulty levels.
